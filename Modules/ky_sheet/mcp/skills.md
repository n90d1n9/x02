# Ky Sheet AI Skills

This document defines the specific skills and competencies that the Ky Sheet AI agent possesses for spreadsheet operations and data management.

## Core Skills

### 1. Spreadsheet Creation & Management

**Skill Level**: Expert

**Capabilities**:
- Create new workbooks with custom configurations
- Add, rename, delete, and reorder worksheets
- Set up sheet properties (tab color, visibility, protection)
- Manage workbook metadata and properties
- Clone sheets and copy between workbooks
- Handle large workbooks with multiple sheets efficiently

**Example Tasks**:
```
- "Create a new workbook with 3 sheets: Data, Analysis, Charts"
- "Rename Sheet1 to 'Q1 Sales'"
- "Copy the formatting from Sheet1 to Sheet2"
- "Hide the calculations sheet"
```

### 2. Data Entry & Manipulation

**Skill Level**: Expert

**Capabilities**:
- Enter single values or bulk data into cells
- Work with different data types (text, numbers, dates, times)
- Auto-fill series and patterns
- Import data from CSV, JSON, and external sources
- Copy, cut, and paste with various options
- Transpose data (rows to columns and vice versa)
- Split and combine cell contents

**Example Tasks**:
```
- "Fill cells A1:A100 with numbers 1 to 100"
- "Import this CSV file into a new sheet"
- "Copy range A1:C10 and paste transposed starting at E1"
- "Split the full names in column A into first and last names"
```

### 3. Formula & Function Mastery

**Skill Level**: Expert

**Capabilities**:
- Write and debug complex formulas
- Use built-in functions across categories:
  - Mathematical: SUM, AVERAGE, COUNT, MIN, MAX, ROUND
  - Statistical: STDEV, VAR, CORREL, FORECAST
  - Logical: IF, AND, OR, NOT, IFS, SWITCH
  - Text: CONCATENATE, LEFT, RIGHT, MID, LEN, TRIM
  - Date/Time: TODAY, NOW, DATE, DATEDIF, NETWORKDAYS
  - Lookup: VLOOKUP, HLOOKUP, INDEX, MATCH, XLOOKUP
  - Financial: PV, FV, PMT, RATE, NPV, IRR
  - Information: ISBLANK, ISNUMBER, ISTEXT, ISERROR
- Create array formulas and dynamic arrays
- Build nested formulas with proper precedence
- Use named ranges in formulas
- Debug formula errors (#DIV/0!, #N/A, #VALUE!, etc.)

**Example Tasks**:
```
- "Calculate the average sales for Q1 excluding weekends"
- "Create a formula to look up employee details by ID"
- "Build a dynamic summary that updates when data changes"
- "Fix the #REF! error in cell F15"
```

### 4. Formatting & Styling

**Skill Level**: Expert

**Capabilities**:
- Apply font formatting (type, size, color, bold, italic)
- Set cell background colors and patterns
- Add and customize borders
- Configure number formats (currency, percentage, date, custom)
- Apply conditional formatting rules
- Create and apply cell styles
- Format tables with headers and banded rows
- Set column widths and row heights (auto-fit or custom)
- Merge and unmerge cells
- Wrap text and control alignment

**Example Tasks**:
```
- "Format the sales column as currency with 2 decimal places"
- "Highlight all values above 1000 in green"
- "Apply alternating row colors to the data table"
- "Make the header row bold with a blue background"
- "Auto-fit all columns to their content"
```

### 5. Data Analysis

**Skill Level**: Advanced

**Capabilities**:
- Sort data by single or multiple columns
- Apply auto-filters and advanced filters
- Create and configure pivot tables
- Generate subtotals and outlines
- Perform what-if analysis (Goal Seek, Scenario Manager)
- Calculate descriptive statistics
- Identify trends and patterns
- Detect duplicates and outliers
- Clean and transform data

**Example Tasks**:
```
- "Sort the data by date, then by sales amount descending"
- "Filter to show only sales above $5000 in Q4"
- "Create a pivot table showing sales by region and product"
- "Find and highlight duplicate entries"
- "Calculate the mean, median, and standard deviation"
```

### 6. Charting & Visualization

**Skill Level**: Advanced

**Capabilities**:
- Create various chart types:
  - Column, bar, line, area charts
  - Pie and doughnut charts
  - Scatter and bubble charts
  - Combo charts
  - Radar, surface, and stock charts
- Customize chart elements (titles, legends, axes, gridlines)
- Format data series and points
- Add trend lines and error bars
- Create sparklines for inline visualization
- Link charts to dynamic data ranges
- Export charts as images

**Example Tasks**:
```
- "Create a line chart showing monthly sales trends"
- "Add a secondary axis for profit margin"
- "Make a pie chart of expense categories"
- "Add a trend line to the scatter plot"
- "Update the chart to include the new Q4 data"
```

### 7. Data Validation & Quality

**Skill Level**: Advanced

**Capabilities**:
- Set up data validation rules (lists, ranges, custom formulas)
- Create drop-down lists for cell input
- Configure input messages and error alerts
- Validate data against external sources
- Check for data consistency and accuracy
- Implement cross-field validation
- Create validation reports

**Example Tasks**:
```
- "Create a dropdown list for status: Pending, Approved, Rejected"
- "Only allow dates between Jan 1 and Dec 31, 2024"
- "Validate that email addresses contain @"
- "Ensure values in column B are greater than column A"
```

### 8. File Operations & Integration

**Skill Level**: Advanced

**Capabilities**:
- Open and save files in multiple formats (XLSX, XLS, CSV, ODS, PDF)
- Import/export data from databases and web services
- Convert between file formats
- Compress and optimize file sizes
- Split workbooks into multiple files
- Merge multiple files into one workbook
- Extract data from protected files (when authorized)

**Example Tasks**:
```
- "Save this workbook as a PDF"
- "Export the data sheet to CSV"
- "Merge these 12 monthly files into one annual report"
- "Import customer data from the SQL database"
```

### 9. Automation & Efficiency

**Skill Level**: Intermediate

**Capabilities**:
- Record and replay common operations
- Create templates for recurring tasks
- Batch process multiple files
- Automate repetitive formatting tasks
- Generate reports on schedule
- Use keyboard shortcuts effectively
- Optimize performance for large datasets

**Example Tasks**:
```
- "Create a template for monthly sales reports"
- "Process all 50 CSV files in this folder"
- "Automate the weekly data refresh and formatting"
- "Generate end-of-month reports for all regions"
```

### 10. Collaboration & Sharing

**Skill Level**: Intermediate

**Capabilities**:
- Add comments and notes to cells
- Track changes and revisions
- Protect sheets and workbooks
- Set user permissions and access levels
- Share workbooks securely
- Resolve conflicts in shared documents
- Create documentation within workbooks

**Example Tasks**:
```
- "Add a comment explaining the calculation in cell D10"
- "Protect the formulas sheet from editing"
- "Share this workbook with view-only access"
- "Document the data sources and update frequency"
```

## Skill Levels Definition

- **Expert**: Can handle complex scenarios, troubleshoot issues, and optimize solutions
- **Advanced**: Proficient in most scenarios, can work independently on typical tasks
- **Intermediate**: Capable of standard operations with some guidance on complex tasks
- **Basic**: Understands fundamentals, can perform simple operations
